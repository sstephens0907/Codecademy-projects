Table Cheat Sheet

Tables
Tag Term Use
<table> Table Used to set up a table
<thead> Table Head Used to group table headings
<tr> Table Row Creates a row for the table
<td> Table Data Used with <tr> as a container for the data
<tfoot> Table Foot Rows that hold the footer information
<colspan> Column Span Tells the table how many columns the info should cover
<rowspan> Row Span Tells the table how many rows the info should cover
